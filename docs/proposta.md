# Proposta

Melhoria proposta: adicionar avaliação temporal automática ao framework VOID com métricas quantitativas reprodutíveis.

Problema identificado: ausência de um módulo padronizado de métricas temporais no fluxo de inferência, resultando em avaliação subjetiva.

Solução implementada: módulo `temporal_metrics.py` + integração nos scripts de inferência.
